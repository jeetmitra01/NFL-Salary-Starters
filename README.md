# NFL Salary Starters

In this project I am going to attempt to analyze median salary's by position, for starters in the NFL 

Starters will be defined using team depth chart and/or snap count, the number of starters is as follows:

#### Offense:
- Wide Receiver - Top 3
- Tight End - Top 2
- Running Back - Top 2
- Quarter Back - Top 1
- Guard - Top 2
- Tackle - Top 2
- Center - Top 1

#### Deffense:
- Corner Back - Top 2
- Safety - Top 2
- Line Backer - Top 3
- Tackle - Top 2
- Defensive End - Top 2

#### Special Teams:
- Kicker - Top 1
- Punter - Top 1
